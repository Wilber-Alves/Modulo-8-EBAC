# Modulo-8-EBAC
 Module 8 Exercise - Introduction to UI - Advanced. Scenes containing more than three with vertical alignments and Grid Layout.

The exercise buttons are:

1) on the title screen: 

 * START GAME button with animations. It activates the scene change to the first level after pressing it.

2) on the HUD, for scenes from levels 1, 2, and 3:

 * HOME button
       
   * CONFIG button - activates a window with buttons for sound effects, music, volume, and screen size
   * MISSION button - activates a window containing missions for the level.
 
    OBS: Buttons that will have functions in the future (to be included)
    * REPLAY
    * SHOP
    * EXIT
 
 * INVENTORY button - shows a grid layout of the character's items with color changes.
 
 * INPUTS button - activates the window containing the game's controls/inputs and their respective functions.
